Mapa Mental de Pruebas de Software

mindmap root((Pruebas de Software)) Manuales Pruebas Exploratorias Explora el sistema sin pasos fijos para descubrir errores inesperados. Pruebas Basadas en Escenarios Sigue escenarios específicos que reflejan situaciones del usuario real. Automatizadas Unitarias Evalúan funciones o métodos individuales de manera aislada para asegurar su correcto funcionamiento. Integración Prueban la interacción entre diferentes componentes del sistema para detectar fallos en la comunicación. Funcionales Verifican que el sistema cumpla con los requisitos y especificaciones funcionales. End-to-End Tests Evaluan el flujo completo de la aplicación desde la perspectiva del usuario final. Regression Tests Aseguran que nuevas modificaciones no rompan funcionalidades existentes. Validación de Sistema Smoke Tests Pruebas preliminares para verificar la estabilidad inicial del sistema. Acceptance Testing Validan que el sistema cumpla con los requisitos del cliente o usuario final. Desempeño Performance Testing Evalúan la velocidad, escalabilidad y estabilidad del sistema bajo diferentes cargas.
